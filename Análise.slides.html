Modelo Autoregressivo Não-linear (NAR)¶

Dado um modelo $\phi$, podemos criar um modelo o modelo:

$$AR_\phi(t+1) = \phi\left([y_{t},y_{t-1},\ldots,y_{t-k-1}]\right)$$

Ou seja, dado um modelo de regress√£o qualquer, iremos alimentar ele com uma janela de $k$ sa√≠das passadas e usar a sa√≠da desse modelo como previs√£o para o tempo atual.

Processo Gaussianos (GP)¶

Um Processo Gaussiano √© uma distribui√ß√£o de probabilidade $p(f)$, onde $f : \mathbb{R}^D\rightarrow\mathbb{R}$.

Seja $\mathbf{f} = [f(x_1), f(x_2),\ldots,f(x_n)]$ tal que $x_i \in \mathbb{R}^D$, perceba que $\mathbf{f}$ √© uma vari√°vel aleat√≥ria. Os parametros que definem um Processo Gaussiano s√£o tais que:

  • Uma fun√ß√£o de m√©dia $\mu(x)$, tal que:
    $$\mathbb{E}(\mathbf{f}) = [\mu(x_1), \mu(x_2),\ldots,\mu(x_n)] = \mathbf{m}$$
  • Uma fun√ß√£o de kernel $k(x,y)$, tal que:
    $$p(\mathbf{f}) \sim \mathcal{N}(\mathbf{m}, \mathbf{K}) \\ \mathbf{K}_{ij} = k(x_i,x_j)$$

O procedimento de regress√£o consiste em montar o seguinte modelo Bayesiano:

$$\begin{align*} f &\sim \mathcal{GP}(\mu,k) \\ y(x) &\sim \mathcal{N}(f(x), \varepsilon) \end{align*}$$

O modelo que usamos nas previs√µes √© a posteriori $p(f\mid \mathcal{D})$ onde $\mathcal{D} = \{(x_i,y_i)\}$ s√£o as nossas observa√ß√µes.

GP-NAR¶

Para usar Processos Gaussiano na tarefa de previs√£o de s√©ries temporais iremos usar a abordagem do NAR sendo $\phi$ um GP. Assim, usamos:

  • $k=4$ que nem na M√©dia M√≥vel;
  • Os primeiros $12\cdot4$ meses como conjunto de treino;
  • Fun√ß√£o de m√©dia linear, $\mu(x) = \mathbf{A}x+b$, onde $\{\mathbf{A}, b\}$ s√£o hiperparametros.
  • Fun√ß√£o de kernel RBF, $k(x,y) = \sigma\cdot\mathrm{exp}\left[-\frac{1}{2}\sum_d w_d(x_{d}-y_{d})^2\right]$ onde $\{\sigma, w\}$ s√£o hiperparametros.

Para escolher os valores dos hiperparametros, fizemos o procedimento usual com modelagem por GP que √© otimizar $\{\mathbf{A}, b, \sigma, w\}$ de tal forma que a verossimilhan√ßa do modelo √© m√°xima.
